~ 核心模块功能

Scanner 扫描器 - 每日自动扫描86个主力合约，筛选非僵尸品种，计算35+技术指标，生成交易信号。

Reasoner 推理引擎 - LLM驱动的信号推理，输出置信度评估和操作建议。包含反叙事约束和概率加权场景分析。

Evolver 因子进化 - 闭环迭代因子发现引擎（Generate -> Execute -> Evaluate -> Gate -> Memory）。

RL 训练系统 - 基于PPO算法的强化学习训练，支持Walk-Forward验证确保泛化能力。

~ 数据源优先级

主数据源：DuckDB（本地缓存，高性能查询）

实时数据：TqSdk（首选，实时行情）

备选数据：Pytdx（通达信直连）

辅助数据：AkShare（基差/季节性/龙虎榜/保证金/宏观）

Fallback机制：当主数据源不可用时，自动切换到备选数据源
